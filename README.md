# go-net | minimal golang terminal based browser

### usage
use it for simple web reading, such as docs or blogs


- F1 - focus on website text
- F2 - focus on website links
- F3 - focus on search bar

arrow keys to navigate
### installation

```shell
git clone https://codeberg.org/suigarra/go-net.git
cd go-net && go build
sudo cp net /usr/bin/gonet
```
